Windows, Unix

The vast majority of shortcuts are the same between operating systems. The main difference is that shortcuts that in Windows use CTRL will, with Mac or Linux machines, use CMD (⌘, the squiggly command key) instead. This little guide assumes you're using a Windows machine, but you can use all of these shortcuts on a Mac or Linux machine just by using, for example, ⌘-S instead of CTRL-S to save something.

Common shortcuts

Opening and saving files

Opening files: In most programs, you can open things with CTRL-O. This will open the menu that appears when you go to File > Open.

Saving files: In most programs, you can save your work with CTRL-S. If you want to rename/create a copy of what you're working on, you can 'Save As' [your filename] with CTRL-SHIFT-S.

Navigating text

Skip through words quickly: hold CTRL and use the arrow keys to skip between words when typing.

Select/highlight: hold SHIFT and either click or use the arrow keys to select some amount of text. You can use CTRL and SHIFT together for this to select full words at once.

Find text: CTRL-F to find text. Once you've entered a word into the search pop-up, you can press ENTER to cycle through all the matches.

Replace text: CTRL-H to replace text. Usually, you can replace things one by one, or all at once ('Replace All').

Copy and paste: CTRL-C to copy a selection, CTRL-V to paste it (and if you want to paste it without formatting, so it matches what you're pasting it into, CTRL-SHIFT-V).

Switching between windows/programs

Switching between windows: You can press ALT-TAB to cycle through all your open windows. This is handy if you just need to check something in another window quickly, or if you need to continually switch between several programs.

Switching between tabs: In most programs that use tabs (internet browsers, text editors...), you can switch between your open tabs by pressing CTRL-TAB.

Using the terminal/command line

In a command line interface (CLI), you input text commands to perform tasks. In Windows 10, Powershell is the built in command line interface. The older version of the windows CLI, which uses DOS, is just called Windows Command Prompt, which you can find by going to Start > Run > 'cmd'. For Mac and a lot of flavors of Linux, it's just called some variation of Terminal. In any case, you can open a terminal window with CTRL-T on Linux (CMD-T on Mac, or CMD-SPACE and search for terminal), and you can open Powershell by hitting the Windows key then searching for it (WINDOWS, 'powershell').

Navigating through folders

Figure out where you are: You can find your current directory (the folder you're 'in' right now) with pwd, which stands for print working directory. This will print out the path or address to the folder you're in.

List everything in the current directory: You can list out what exists in the current directory with ls.

Getting around: To change directories, you can cd into a folder. You can cd into a folder contained in the directory you're in, or give the full address. For example, if you're already in PTB/mosel then cd assim and cd PTB/mosel/assim will do the same thing. To go one level up, you can use the shortcut cd .. (so if you're in PTB/mosel and you enter cd .., you'll end up in PTB).

You can also drag things in from finder/whatever file explorer you have to the terminal to autotype the path. In Finder specifically, you can also open a folder into from the terminal (e.g. navigate to /Volumes/ptb then in the shell open .)

Using programs

You can use programming languages like Python, Octave, and R directly in the command line interface. Simply type the name of the language: python3 for Python 3, octave for Octave, and so on.

Program-specific shortcuts

Audacity

The regular shortcuts of cut (CTRL-X), copy (CTRL-C), and paste (CTRL-V) apply. You can move pieces of a track around with the Time Shift tool.

Playing/stopping: Press SPACE to play (starting from where you last moved the cursor to), and press SPACE again to stop playing. This won't actually pause the audio -- the next time you press space, you'll just start from where your cursor is again. You can loop whatever you've highlighted with SHIFT-SPACE, so that it'll just keep playing that selection until you hit the space bar again.

Zooming in: CTRL-1 to zoom in a pre-set amount.

Zooming out: CTRL-3 to zoom out a pre-set amount.

Zooming to selection: To zoom to the part you've highlighted, CTRL-E.

Find zero crossings: If you've already clicked near where you want to find a zero crossing, just press Z. If you've highlighted a selection, pressing Z will move both boundaries towards the nearest zero crossing.

Use labels: Use CTRL-B to create a label (either at the point you clicked, or over the region you've highlighted). You can name the label (you can copy/paste within the label field) so you know what's going on there. You can export just the parts of interest by labeling them, then File > Export Multiple (split files based on label).
