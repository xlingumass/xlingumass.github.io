    Phonetics Lab

    Research in the Phonetics Laboratory is focused on three questions:

    1. How do listeners perceive speech sounds?
    2. How is their perception of speech sounds influenced by the contexts in which those sounds occur?
    3. How does the perception of speech sounds influence their phonological patterning?

    Presently, our work is concentrated on three more specific versions of these questions:

    1. When does a listener's knowledge that a particular string of sounds is a word influence their perception of an ambiguous sound in that string?
    2. Does the order of two sounds in a string influence how they influence one another's perception?
    3. Does a listener's phonotactic knowledge have second as well as first order effects on the perception of speech sounds?

    Eyelab (N442)

    We use an SR Research Eyelink 1000 with a tower mount system. Our lab is setup using the following parameters:

    • The screen is located XX inches away from the mount, always.
    • The size of the screen is XX by XX, and we present our experimental stimuli using 11pt Monaco font.
    • Given these parameters, a single degree of visual angle subtends XXX characters, on average.

    Sideeye

    Sideeye is a python package for processing eyetracking data (as DA1 files), created by Amanda Doucette, an alum of the CSL Lab.

    We use R for analysis.

    From Adrian Staub and Chuck Clifton at the UMass Eyetracking Lab:

    Scripter2

    Scripter2 is a Perl program which takes a tab-delimited file of stimuli as input (see Fig X) and returns a .script file for use in the Eyetrack program.

    Robodoc

    This python program does automatic trial exclusion from .asc files based on blink/track losses and incorrect display changes, producing .DA1 files.
